# 🌌 Mai QQ Tools (`mai_qq_tools`)

> 赋予 AstrBot 灵魂深处的拟人化物理交互能力。

`mai_qq_tools` 是一个为 [AstrBot] 打造的全局函数工具扩展插件。它脱胎于庞大的 `astrmai` (https://github.com/0d00-Ciallo-0721/astrbot_plugin_astrmaimai)认知生态，经过深度解耦与提纯，将 7 个最高频的 QQ 专属物理互动动作开放给全局 LLM 调用。

0 数据库依赖，即插即用，彻底打破大语言模型只能“打字”的壁垒。

---

## ✨ 核心特性与工具集 (Core Features)

插件通过 `@filter.llm_tool` 向 AI 上下文注册了以下 7 大物理互动能力：

1. 🎯 **主动 @ 构造 (Construct At)**
   - **能力**：允许 AI 主动在回复中 `@` 特定群友。
   - **亮点**：内置防抖与去重机制，严禁 AI `@` 自己。
2. 👉 **主动戳一戳 (Proactive Poke)**
   - **能力**：调用底层 API，向目标用户发送双击头像的“戳一戳”动作。
3. 🎭 **人类本质复读机 (Meme Resonance)**
   - **能力**：当 AI 观察到群友在“保持队形”刷屏时，可以调用此工具强行中断思考，执行 1:1 完美复读。
4. 🤫 **悄悄话跨界 (Space Transition)**
   - **能力**：在群聊中触发，AI 会偷偷向指定群友发送私聊，并在群内留下掩护话语（如：“看私聊”）。
   - **黑科技**：自带**跨界记忆溯源**。用户在私聊中回复时，系统会自动抓取跳出前的 5 条群聊记录注入 AI 潜意识，实现无缝接话。
5. 🛑 **手滑撤回与找补 (Regret & Withdraw)**
   - **能力**：AI 会真实发送一条冲动消息（如说漏嘴），**1.5 秒后自动跨平台撤回**，并紧接着发送找补/掩饰话语。
6. ✨ **贴表情回应 (Message Reaction)**
   - **能力**：AI 可以用 QQ 官方小表情（如爱心、流汗、敲打）直接贴在用户的消息下方作为回应。
7. 👍 **狂踩名片赞 (Proactive Like)**
   - **能力**：AI 会冲到用户的 QQ 个人主页，连续点赞 50 次，并将是否被空间权限拦截的真实物理反馈作为后续聊天的谈资。


### 2. 前置要求
- **AstrBot 版本**：建议 `>= v4.5.7`（依赖新的全局 Tool 注册 API）。
- **协议端要求**：高度依赖 **OneBot v11** 协议标准（推荐使用 `Napcat`）。在其他协议下可能失效。

## 🧪 测试指南 (Testing / Prompts)

在群聊或私聊中，你可以使用以下话术测试大模型是否能正确调用对应工具：

- **测试【贴表情/戳一戳】**
  > *"你觉得群主可爱吗？觉得的话去戳一下他，或者给他刚才那句话贴个爱心的表情。"*
- **测试【手滑撤回】**
  > *"你能不能假装不小心对我表白，然后光速撤回，再随便找个借口掩饰一下？"*
- **测试【悄悄话跨界】**（在群聊中发送）
  > *"我有个秘密想告诉你，但群里人太多了，你能不能偷偷私发我？"*
  > *(等待 AI 发送私聊后，切到私聊窗口说：“好了，群里人看不到了，你说吧”，测试上下文是否连贯。)*
- **测试【狂点赞】**
  > *"今天心情不好，你能不能去我主页给我狂点一波赞安慰一下我？"*
